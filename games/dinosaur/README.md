# 🦕 超級進階小恐龍遊戲 v2.0

一個功能豐富的小恐龍跳躍遊戲，採用模組化設計，易於維護和擴展。

## 🎮 遊戲特色

### 🎯 四種難度等級

- **簡單 (Easy)** - 適合新手，慢節奏遊戲
- **中等 (Medium)** - 標準難度，平衡的挑戰
- **困難 (Hard)** - 快節奏，需要高度技巧
- **噩夢 (Nightmare)** - 超極速+重力異常+螢幕閃爍

### 🦕 恐龍特殊技能

- **跳躍** - 基本跳躍 + 二段跳
- **蹲下** - 避開飛行障礙物
- **衝刺** - 快速前進（有冷卻時間）
- **護盾** - 短時間無敵保護

### 🌵 多樣化障礙物

- **基礎障礙物**: 仙人掌、石頭、飛鳥
- **進階障礙物**: 雙重、上下移動、隱形、爆炸、裝甲
- **特殊機制**: 不同障礙物需要不同的應對策略

### 🖥️ 顯示功能

- **全螢幕支持** - F11 切換全螢幕模式
- **動態解析度** - 自動適應不同螢幕尺寸
- **視窗縮放** - 支援視窗大小調整
- **動態 UI** - 根據螢幕大小自動調整介面

### ⚡ 噩夢模式特效

- **重力反轉** - 重力可能突然反轉
- **控制反轉** - 操作控制可能顛倒
- **螢幕閃爍** - 視覺干擾效果
- **能力故障** - 技能可能暫時失效

## 🕹️ 操作說明

### 基本操作

- `↑` / `空白鍵` - 跳躍（可二段跳）
- `↓` / `S鍵` - 蹲下
- `X鍵` - 衝刺（有冷卻時間）
- `Z鍵` - 護盾（短時間無敵）

### 系統操作

- `ESC鍵` - 返回主選單
- `F11鍵` - 切換全螢幕模式
- `Alt+F4` - 退出遊戲

## 🎯 遊戲技巧

### 基礎生存

1. **觀察障礙物** - 不同障礙物需要不同策略
2. **善用技能** - 合理使用衝刺和護盾
3. **保持節奏** - 找到適合的跳躍和蹲下時機

### 進階技巧

1. **連擊系統** - 連續避開障礙物可獲得額外分數
2. **二段跳** - 在空中可以進行第二次跳躍
3. **預判能力** - 提前觀察障礙物類型和位置

### 噩夢模式生存指南

1. **適應重力** - 重力反轉時快速調整策略
2. **控制反轉** - 注意操作可能顛倒
3. **保持冷靜** - 螢幕閃爍時不要慌張
4. **技能管理** - 能力故障時依靠基本操作

## 📊 計分系統

### 基礎計分

- 成功避開障礙物: +10 分
- 走過石頭障礙物: +5 分
- 蹲下避開飛鳥: +10 分
- 護盾阻擋攻擊: +20 分

### 難度倍率

- 簡單: x1
- 中等: x1.5
- 困難: x2
- 噩夢: x4

### 連擊獎勵

- 連續避開障礙物可累積連擊數
- 連擊越高，獲得的額外分數越多

## 🔧 開發說明

### 模組化設計

遊戲採用模組化設計，各個系統分離：

- **game_config.py** - 統一管理遊戲配置
- **dinosaur.py** - 恐龍角色的所有邏輯
- **obstacles.py** - 障礙物系統和管理器
- **menu_system.py** - 選單界面和交互
- **game_engine.py** - 協調所有系統運作

### 擴展性

- 易於添加新的障礙物類型
- 可以輕鬆調整難度參數
- 支援添加新的特殊效果
- 模組化設計便於功能擴展

## 🎨 自定義

### 顏色主題

在 `game_config.py` 中修改 `get_color_palette()` 函數來自定義顏色。

### 難度調整

在 `game_config.py` 中修改 `DIFFICULTY_SETTINGS` 來調整各難度的參數。

### 新增障礙物

在 `obstacles.py` 中的 `Obstacle` 類別添加新的障礙物類型。
